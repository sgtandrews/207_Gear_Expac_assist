# 207_Gear_Expac
Private Git for 207’s Gear Expansion mod

# Mission Statement
This gear expansion aims to add additional items to the 207 limited arsenal for players to use.
This may include bespoke items, such as new survival fatigues, ghillie suits, or items with special functions.
The initial stages of this project seek to add a limited number of new items and gather community feedback.
The project should respect player filesize limitations, and not overly bloat existing mod sizes.

# Changes
## V0.1
- Implemented one set of Crye G3 uniforms from TFB modset.
- Altered patches to include bespoke 207 and Australia flag shoulder patches.
- Altered config data to ensure uniform carry load and item weight are consistent with ADFRC standard.
- Recreated ACE Arsenal Extended config to include new variants.
